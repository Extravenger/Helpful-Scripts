Token Duplicator that can be used with Execute-Assembly in CS or Havoc.

Compile with Mono-Devel on Linux. 

mcs tokenduplicatorC#.cs /out:token.exe

Usage:

token.exe -p PROGRAM -a "ARGS" (with quotes)
